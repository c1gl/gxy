Brief Introduction of Golden State Warriors

金州勇士队（Golden State Warriors）于1946年成立并加盟BAA（1949年加盟NBA），是一支属于美国加利福尼亚州旧金山市的职业篮球队，是美国职业篮球联赛（NBA）西部联盟太平洋赛区的一部分。 勇士队主场原来在宾夕法尼亚州的费城，1962年才搬到加州；由于加州的昵称是“金州” ，故取名“金州勇士” 。它是最早加盟NBA的11支球队之一；在NBA历史上，曾七次夺得总冠军 ，NBA的第一次总冠军得主就是勇士队；威尔特·张伯伦效力勇士队期间缔造了NBA单场最高分纪录100分。 在唐·尼尔森成为球队教练后，勇士队成为了一支可以为季后赛席位展开争夺的劲旅，2006-07赛季季后赛第一轮更是淘汰常规赛冠军小牛队创造NBA史上第3个“黑八奇迹”。 2014年5月，史蒂夫·科尔成为勇士队主教练。2014-15赛季，勇士以67胜15败排名西部联盟第一。2015年6月17日，NBA总决赛第六场，勇士客场以105-97再胜骑士，总比分4-2夺取总冠军。这是勇士队在1974-75赛季以来的第一个冠军。2015-16赛季，勇士队在NBA常规赛中取得73胜9负的成绩，打破公牛在1995-96赛季创下的纪录（72胜10负），成为NBA历史单赛季常规赛战绩最好的球队。2017年6月13日，勇士队以16胜1负的季后赛战绩，超越2000-01赛季的湖人（15胜1负），成为NBA历史上胜率最高的夺冠球队。2017-18赛季，勇士队在NBA总决赛总比分4-0战胜克里夫兰骑士队完成卫冕。2021-22赛季，勇士队在NBA总决赛大比分4-2战胜波士顿凯尔特人队夺得队史第七个总冠军 022年休赛期，勇士签下莱恩·罗林斯 、唐特·迪温琴佐 、莱斯特·基诺内斯 、瑞安·罗林斯 、贾玛考·格林 、迈克·麦克朗、帕特·斯宾塞和特雷维恩·威廉姆斯 。 由于小佩顿的离去和新援迪温琴佐的伤缺，勇士的外线防守质量严重下滑。2022-23赛季开赛前10场比赛，勇士仅收获3场胜利，排名西部第13位，成为历史首个客场开局0胜6负的卫冕冠军。为了提升球队的防守质量，勇士在交易截止日前从开拓者换回了小佩顿。常规赛最后10场比赛，勇士收获了8场胜利，最后3个客场也取得了2胜1负的战绩，最终以44胜38负的战绩位居常规赛西部第六。季后赛首轮，勇士历经抢七大战以大比分4-3淘汰国王，晋级第二轮。西部半决赛，勇士大比分2-4被湖人淘汰，连续28次季后赛系列赛在客场至少拿到1场胜利的纪录遭遇终止。这是勇士在科尔执教以来首次输掉除总决赛之外的系列赛。 2023年休赛期，勇士送出普尔、瑞安·罗林斯、小帕特里克·鲍德温，得到克里斯·保罗 、科里·约瑟夫 、达里奥·沙里奇 、布兰丁·波杰姆斯基 、鲁迪·盖伊 、杰罗姆·罗宾逊 、罗德尼·麦格鲁德 ；在选秀大会中选中特雷斯·杰克逊-戴维斯。10月，WNBA官方宣布，勇士将正式在湾区拥有一支新的WNBA球队，在2025赛季开始参加WNBA联赛 。
